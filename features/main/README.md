# 기능들

  * 각 직업군별 랭킹을 기준으로한 통계
    * Top 100, 200, 500, 500 부터는 랜덤한 페이지 사용할수도
  * 던파 사이트 랭킹 업데이트
    * 외부 랭킹 사이트 스크레이핑
  * 외부 랭킹 사이트 직업군 별 모험가 총합
    * 그냥 `마지막 페이지 모험가 수 + (마지막 페이지 - 1) * 10`
  * 던파 모험가 이름 목록 업데이트
    * 공식 던파 게시판, 외부 랭킹 사이트 스크레이핑
    * 던파 사이트 랭킹 업데이트 시 자동으로 업데이트 됨
